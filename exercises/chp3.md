# 3장 연습문제

2. 파이썬은 2020년 이후 가장 중요한 프로그래밍 언어로 급부상했습니다. 그 이유를 설명하세요.
    - 파이썬은 문법이 쉽고 머신러닝, 딥러닝 라이브러리, 데이터 분석 등 AI / 데이터 시대의 핵심 언어로
      자리 잡았고 범용성이 좋아 핵심 언어로 떠올랐다.

3. 마이크로파이썬을 이용해 IoT 디바이스용 프로그램을 작성할 때 얻을 수 있는 이점을 정리하세요.
    - 1. 경량 + 저사양 디바이스에 적합하다.(ESP32 같은 MCU에서 동작)
    - 2. 개발 속도가 빠르다.(컴파일 없이 바로 실행해 테스트가 빠르다.) 
    - 3. 파이썬 문법을 그대로 사용한다.
    - 4. 하드웨어 제어 라이브러리를 제공한다.(센서 / 액추에이터 제어 코드가 간단함)
    - 5. 네트워크 기능이 내장되어있다.(Wi-Fi, MQTT, HTTP 통신을 쉽게 구현할 수 있다.)
